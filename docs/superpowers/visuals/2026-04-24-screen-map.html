eSIM Market: 3M Architecture Screen Map

Phân rã chức năng cho 2 vai trò chính dựa trên kiến trúc 3 lớp

🏢 3M Admin (Chủ sàn)

  • ● Command Center: Dashboard tổng quan vĩ mô.
  • ● Upstream Hub: Quản lý Supplier & Sync API.
  • ● Marketplace Matrix: Inventory & Pricing Engine.
  • ● Downstream Control: Quản lý Agent & API Keys.
  • ● Master Finance: Ví tổng, đối soát (Reconciliation).

👤 Agent (Đại lý)

  • ● Sales Dashboard: Theo dõi biến động kinh doanh cá nhân.
  • ● eSIM Store: Giao diện mua sắm (Retail/Wholesale).
  • ● My Assets: Quản lý đơn hàng & mã eSIM đã mua.
  • ● Wallet Hub: Nạp tiền, lịch sử chi tiêu.
  • ● Dev Studio: Tài liệu API & Webhook (cho sub-agent).
